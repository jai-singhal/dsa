C program to do binary Search in the file using id(first 6 characters of each line) without using fopen, fread.

lseek: To seek in the file

**Time Complexity**: O(log(n))
where n = no. of lines in file.


Execute by-

`
gcc newCode.c
`

And then run

`
./a.out nanpa 100017
`
